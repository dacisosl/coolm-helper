📥 최근 메시지함에서 일정 등록하기

펭귄 메뉴의 쪽지 목록을 누르면 최근 쪽지에서 찾은 일정 후보가 한 번에 보여요. 왼쪽에서 고르고 오른쪽에서 확인·수정 후 등록. 안읽은 쪽지(●)는 전부 포함되고, 등록한 항목은 ✓ 연두색으로 표시됩니다.

주요 기능 ①

🗓 캘린더 · 할일

등록한 일정은 전부 여기서 확인·관리합니다. 인터넷 없이도 돼요.

  • 날짜 아래 파란 배지 = 그날 일정 개수, 빨간 배지 = 중요도 '높음' 포함
  • 날짜 클릭 → 오른쪽에 그날 일정 목록
  • 일정 클릭 → 아래로 펼쳐지며 제목·일시·중요도·메모 수정 → [저장]
  • 마감 체크한 일정 = 할일 — 알림 대상이 됩니다
{{SHOT:calendar}}
수정하면 구글도 같이. 구글 캘린더 모드를 켠 경우, 여기서 고치거나 지우면 구글 사본도 함께 바뀝니다.
주요 기능 ②

🔔 시작 알림

컴퓨터를 켜면 펭귄이 먼저 알려줍니다.

  • 프로그램이 켜지면 펭귄 머리 위에 말풍선이 떠요
  • 내용: 마감 7·3·1일 전 할일 → 오늘 일정 N건 순서
  • 말풍선을 클릭하면 다음 알림, 마지막 클릭에 닫힘
  • 한 번 보면 다시 켜기 전까진 반복되지 않아요

안심하고 쓰세요

🔒 개인정보는 이렇게 지켜집니다

학생 정보를 다루는 만큼, 구조부터 안전하게 만들었습니다.

  • 쪽지 분석·일정 추출은 전부 선생님 PC 안에서만 처리됩니다
  • 쪽지 원문·학생 명단은 절대 밖으로 나가지 않습니다
  • 빨간 글씨 = 전화번호·이름으로 보이는 부분 표시 — 자동으로 지우진 않으니 등록 전에 직접 확인·삭제하세요
  • 구글 모드를 켠 경우에도 나가는 건 확인을 거친 제목과 날짜뿐
  • 쿨메신저는 읽기만 해요 — 쪽지·안읽음 상태가 바뀌지 않습니다
학생 명단 등록(선택): 설정 → 개인정보에서 명단 파일에 학생 이름을 넣어두면 그 이름들이 자동으로 빨갛게 표시됩니다.

❓ 문제 해결

대부분 여기서 해결됩니다.

  • 설치할 때 파란 경고("Windows의 PC 보호")가 떠요 → 개인 제작 프로그램이라 뜨는 정상 안내예요. 추가 정보 → 실행.
  • "보고 있는 쪽지를 찾지 못했습니다" → 쿨메신저에서 쪽지 창을 연 상태에서 더블클릭하세요. 그래도 안 되면 [클립보드에서 가져오기].
  • 펭귄에 주황색 D가 떠 있어요 → 데모 모드입니다. 설정 → 데이터에서 끄세요.
  • 날짜가 이상하게 잡혔어요 → 등록 전에 날짜 버튼을 눌러 달력에서 고르면 됩니다. 이상한 사례는 알려주시면 다음 버전에서 개선돼요!
  • 그 밖의 문제 → 배포해 주신 선생님께 문의해 주세요.
