# COOL-비서 — 설치 안내

## 설치

1. `CoolmHelper-Setup.exe`를 실행하고 안내를 따르세요. (관리자 권한 불필요)
2. 설치 중 Windows SmartScreen이 "인식할 수 없는 앱" 경고를 띄울 수 있습니다.
   서명 없는 개인 제작 프로그램이라 나타나는 정상적인 경고입니다 —
   **추가 정보 → 실행**을 누르면 됩니다.
3. 첫 실행 시 쿨메신저 메시지함 위치를 자동으로 찾습니다
   (`%LOCALAPPDATA%\CoolMessenger\Memo`). 별도 설정이 필요 없습니다.

## 첫 사용

1. 화면 우측 하단에 작은 위젯이 뜹니다. 드래그로 위치를 옮길 수 있습니다.
2. **[📅 일정 등록]** — 가장 최근 쪽지 10개에서 일정 후보를 찾아 보여줍니다.
   (창 위쪽에서 50개·100개로 늘릴 수 있습니다.)
   빨간 글씨는 전화번호·이름으로 보이는 부분입니다. 제목·일시를 확인/수정 후
   [등록]을 누르세요.
3. **[🗓 캘린더 / 할일]** — 등록한 일정을 월간 캘린더와 할일 목록으로 봅니다.
   인터넷 연결이 필요 없습니다.

## 학생 명단 설정 (권장)

설치 폴더의 `students.txt.example`을 복사해 같은 위치에 `students.txt`로 저장하고,
우리 반 학생 이름을 한 줄에 하나씩 적으세요. 일정 제목에 학생 이름이 들어가면
자동으로 ○○○ 처리됩니다. 이 파일은 어디에도 전송되지 않습니다.

## 구글 캘린더 연동 (선택)

기본 기능은 구글 계정 없이 전부 동작합니다. 구글 캘린더에도 올리고 싶으면
`구글연동설정.md`를 참고하세요.

## 문제 해결

- "메시지 DB를 찾을 수 없습니다" → 쿨메신저가 설치·로그인되어 있는지 확인.
  경로가 다르면 설치 폴더 `config.json`의 `memo_dir`를 수정하세요.
- 쿨메신저 업데이트 후 동작하지 않으면: 쿨메신저 메시지 관리함에서 엑셀로
  내보내기 하면 프로그램이 자동으로 그 파일을 읽습니다(Plan B 모드).

## 삭제

제어판 → 프로그램 제거. 등록한 일정(`store/events.json`)과 학생 명단은
설치 폴더에 남으니 필요하면 삭제 전에 백업하세요.
