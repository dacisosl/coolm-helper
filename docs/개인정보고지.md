# 개인정보 처리 안내

**COOL-비서**(구 쿨메신저 일정 도우미)는 다음 원칙으로 동작합니다.

## 이 프로그램이 하는 일

- 쿨메신저 메시지함(내 PC의 로컬 파일)을 **읽기 전용**으로 접근합니다.
  원본 파일을 수정·이동·삭제하지 않습니다.
- 버튼을 눌렀을 때만 읽습니다. 백그라운드 감시나 자동 수집이 없습니다.
- 메시지에서 날짜·시간을 찾는 파싱, 개인정보 탐지·마스킹은
  **전부 내 PC 안에서** 정규식과 명단 대조로 처리됩니다. AI·외부 서버를 쓰지 않습니다.

## 내 PC 밖으로 나가는 데이터

- 기본 상태: **없음.** 일정·할일은 내 PC(`store/events.json`)에만 저장됩니다.
- 구글 캘린더 연동을 직접 켠 경우: 사용자가 미리보기 화면에서 확인·수정한
  **일정 제목과 시작·종료 일시만** 구글에 전송됩니다.
  쪽지 원문, 보낸 사람, 첨부파일명은 전송되지 않습니다.

## 개인정보 표시

전화번호, 주민등록번호, 호칭이 붙은 이름(○○ 학생, ○○님 등), 학생 명단
(`students.txt`)·교직원 명단과 일치하는 이름은 **빨간 글씨로 표시**됩니다.
자동으로 지우지는 않으며, 지울지 여부는 등록 전 미리보기에서 사용자가
직접 편집해 결정합니다. 자동 탐지가 완벽하지 않을 수 있으므로,
**등록 전 제목을 반드시 확인하세요.**

## 데이터 보관

- 등록한 일정: 설치 폴더의 `store/events.json` (로컬)
- 학생 명단: 설치 폴더의 `students.txt` (로컬, 사용자가 직접 작성)
- 구글 로그인 토큰(연동을 켠 경우): `calendar_sync/token.json` (로컬)
- 프로그램 삭제 시 위 파일을 함께 지우면 모든 데이터가 제거됩니다.

문의: 프로그램 배포자에게 연락하세요.
